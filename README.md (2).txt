# Random Quote Generator

## Автор
Иванов Иван

## Описание
Random Quote Generator — это GUI-приложение для генерации случайных цитат из предопределённого списка. Приложение позволяет:
- Получать случайные цитаты по нажатию кнопки
- Сохранять историю просмотренных цитат
- Фильтровать цитаты по автору или теме
- Сохранять и загружать историю в формате JSON
- Добавлять новые цитаты с проверкой корректности ввода

## Примеры использования

### Основные функции:
1. **Генерация случайной цитаты**
   - Нажмите кнопку "Сгенерировать цитату"
   - В главном окне отобразится случайная цитата с указанием автора и темы

2. **Просмотр истории**
   - Откройте вкладку "История"
   - Увидите список всех ранее сгенерированных цитат
   - История сохраняется между сессиями

3. **Фильтрация цитат**
   - Во вкладке "Фильтры" выберите автора или тему
   - Нажмите "Применить фильтр"
   - В истории будут отображаться только соответствующие цитаты

4. **Добавление новых цитат**
   - Перейдите во вкладку "Добавить цитату"
   - Заполните поля: текст, автор, тема
   - Нажмите "Добавить"
   - Пустые поля будут заблокированы системой валидации

### Возможные тесты:
1. **Тест генерации**: нажать кнопку генерации 10 раз подряд, убедиться что цитаты разные
2. **Тест истории**: сгенерировать 5 цитат, проверить что все они сохранились в истории
3. **Тест фильтрации**: выбрать конкретного автора, убедиться что отображаются только его цитаты
4. **Тест сохранения**: закрыть и открыть приложение, проверить что история сохранилась
5. **Тест валидации**: попытаться добавить цитату с пустым текстом - должно появиться предупреждение

## Технические особенности
- Использует модуль `random` для случайного выбора
- История сохраняется в файл `quotes_history.json`
- Поддерживает форматы JSON для импорта/экспорта данных
- Валидация ввода предотвращает добавление некорректных данных
- Проект размещён в Git-репозитории с полной историей коммитов

## Требования
- Python 3.7+
- Модули: tkinter, json, random
- Git для контроля версий

## Git-репозиторий
Проект размещён в Git с следующими основными коммитами:
- `init`: начальная структура проекта
- `feat: basic gui`: базовый интерфейс
- `feat: quote generation`: функционал генерации цитат
- `feat: history and filters`: история и фильтрация
- `feat: json save/load`: сохранение в JSON
- `feat: input validation`: валидация ввода
