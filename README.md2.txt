# GitHub User Finder (Поиск пользователей GitHub)

## Автор
Иван Иванов

## Описание
GitHub User Finder - это GUI-приложение для поиска пользователей GitHub через официальное API. Приложение позволяет искать пользователей, просматривать их основную информацию и сохранять понравившихся пользователей в избранное.

## Функциональность
- Поиск пользователей GitHub по логину или имени
- Отображение основной информации о пользователях
- Добавление пользователей в избранное
- Сохранение избранных пользователей в локальный JSON-файл
- Удаление пользователей из избранного
- Простой и интуитивно понятный графический интерфейс

### Основные эндпоинты:
1. **Поиск пользователей**

GET /search/users?q={query}&per_page={count}

Пример: `https://api.github.com/search/users?q=torvalds&per_page=10`

2. **Получение информации о конкретном пользователе**

GET /users/{username}

Пример: `https://api.github.com/users/torvalds`

### Ограничения API
- Анонимные запросы: до 60 запросов в час
- С аутентификацией: до 5000 запросов в час
- Для увеличения лимитов рекомендуется использовать персональный токен доступа
